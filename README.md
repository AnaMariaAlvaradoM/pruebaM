# Master IA — Módulo 1

Script de análisis de datos desarrollado en el Módulo 1 del Máster en IA & Data Science — Dev Senior Code.

## ¿Qué hace?
Procesa un dataset numérico y muestra un resumen con promedio, máximo y total de registros.

## ¿Cómo ejecutarlo?
```bash
python analisis.py
```

## Tecnologías
- Python 3.x
